About LTSD Analyzer

This tool analyzes Long-Term Supplier Declarations (PDF) and extracts materials, Commodity Code (8 digits), COO (ISO‑2), and Preference (Yes/No). It flags cases where Preference = Yes but COO is not an EU country.

  • Upload the LTSD PDF using the Upload section.
  • Use Filters to narrow results (Material, Description, Commodity, COO, Status, Preference).
  • Toggle “Only Preference Yes & COO non‑EU” to find potential inconsistencies.
  • Export results as CSV or print the table for reviews.

Note: Parsing relies on positions; if a new layout fails, share a sample to improve detection.
